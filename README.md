
## 🗂 Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/LailsonDaSilva/BackendIbelieve.git

    # Entrar no diretório
    $ cd BackendIbelieve

    # Instalar as dependências
    $ npm install

    # Iniciar o projeto
    $ npm run dev

```
